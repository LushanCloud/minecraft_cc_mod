# Schematic Builder - 蓝图建造脚本

通过 turtle-schematic-builder 生成的自动建造脚本。

## 文件说明

| 文件 | 说明 |
|------|------|
| `build.lua` | 主建造脚本（5x5x5立方体框架） |
| `lib.lua` | 核心库 |
| `inv.lua` | 物品栏管理 |
| `ui.lua` | 界面显示 |
| `stateful.lua` | 状态化包装器（支持断点续传） |

## 状态化建造（推荐）

使用 `stateful.lua` 可以让海龟在中断后恢复建造：

### 启用方法

将构建脚本中的：
```lua
local lib = require("lib")
```
改为：
```lua
local lib = require("stateful")
```

### 功能特性

- **自动保存进度**：每步操作后保存状态
- **断点续传**：重新运行时从中断处继续
- **燃料等待**：燃料耗尽时暂停而非报错
- **状态文件**：保存在 `/.states/` 目录

## 使用方法

### 1. 上传所有文件到海龟

在海龟中运行以下命令下载所有文件：

```lua
-- 方法1: 手动创建
edit lib
-- 粘贴 lib.lua 内容

edit inv
-- 粘贴 inv.lua 内容

edit ui
-- 粘贴 ui.lua 内容

edit build
-- 粘贴 build.lua 内容
```

### 2. 补给站布局

**重要**：海龟背后需要有补给箱子！

```
          [北/前方] ← 建造方向
              ↑
              |
[燃料箱] ←   🐢   → [空]
   西        |        东
              ↓
          [石头箱] ← 补给箱（海龟身后）
            南/后方
```

**补给箱需要**：
- 44 个石头方块（或其他建筑方块）
- 燃料（煤炭/木炭）放在燃料箱

### 3. 运行脚本

```lua
build
```

### 4. 工作流程

脚本会自动：
1. 转身面向补给箱
2. 加载燃料
3. 清空背包
4. 从箱子取 44 个石头
5. 转回来开始建造
6. 完成后返回

## 注意事项

1. **海龟需要燃料** - 至少需要 46 燃料值
2. **补给箱在身后** - 初始时海龟背对补给箱
3. **方块类型** - 脚本需要 `minecraft:stone`，如需其他方块需修改
